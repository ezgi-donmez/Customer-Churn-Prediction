# Customer Churn Prediction

This project focuses on predicting customer churn for an e-commerce company using machine learning models.

## Files
- `notebooks/EzgiDonmez.ipynb`: main analysis and modeling notebook
- `report/EzgiDonmez.docx`: written report
- `data/E_Commerce_Dataset.xlsx`: dataset

## Project Steps
1. Data cleaning and preprocessing
2. Exploratory data analysis
3. Model training
4. Model evaluation
5. Feature importance analysis
6. Business insights

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting
